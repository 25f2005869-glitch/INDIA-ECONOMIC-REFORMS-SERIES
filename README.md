# 🇮🇳 India Growth & Development Dashboard

A long-term Python-based Economic Analytics Repository focused on analyzing India’s major economic reforms, financial crises, market revolutions, banking transformations, taxation systems, globalization, fiscal discipline, and development journey using:

- Python
- Data Analytics
- Statistics
- Economic Visualization
- Historical Economic Research

This repository combines Economics + Finance + Statistics + Governance + Data Science into interconnected analytical projects.

```bash

```

## 📌 Go To Repository Folder

```bash
cd india-growth-dashboard
```

## 📌 Install Dependencies

```bash
pip install -r requirements.txt
```

## 📌 Run Analysis

```bash
python analysis.py
```

## 📌 Run Dashboard

```bash
python dashboard.py
```
